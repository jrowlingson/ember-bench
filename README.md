# ember-bench

A simple tool for benchmarking various ember-cli commands.

### Installation

```sh
/bin/zsh -c "$(curl -fsSL https://raw.githubusercontent.com/jrowlingson/ember-bench/master/install.sh)"
```

### Usage

```sh
cd /your/project/root
ember-bench
```

### Options

* Version info
`--version` or `-v`
